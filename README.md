# Tic-Tac-Toe-game
This is a simple and easy to use Tic Tac Toe game build using HTML, CSS, and JavaScript.The game allows two players to take turns marking X and O in a 3Ã—3 grid. The first player to align three of their marks horizontally, vertically, or diagonally wins the game. If all cells are filled and no one wins, the game ends in a draw.
